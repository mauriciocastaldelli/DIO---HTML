  1. O que é HTML?

    2. Hyper Text Markup Language

    Trata-se de uma linguagem de marcação.

    Linguagem de Marcação ≠ Linguagem de Programação

    Descreve para o navegador a estrutura de uma página.

  2. O que são tags?

    3. São elementos que dizem ao navegador como exibir conteúdos.

    Tags geralmente seguem o modelo: <tag inicial> conteudo <tag de fechamento>

    No entanto, existem algumas tags vazias, ou seja, que não possuem a tag de fechamento.

    Elas podem ter atributos que incrementam alguma informação sobre elas.

  3. Atributos

    4. Todo elemento HTML pode ter atributos.

    Atributos fornecem informações complementares para os elementos.

    Sempre estão localizados na tag inicial.

    A sintaxe é: nome="valor"

    Atributos comuns:
    • width
    • height
    • href
    • id
    • class

  6. Links

    Para sites externos:

    <a href="http://url.com.br"> Conteúdo que ativará o link </a>

    Para sites internos:

    <a href="/caminhodoarquivo"> Conteúdo que ativará o link </a>

    Para partes do seu site:

    <a href="#idDaTagAlvo"> Conteúdo que ativará o link </a>

    Atributos comuns:
    • href="caminho do link"
    • title="dica para o usuário"
    • target="_blank para novas abas"
    • target="_self para na mesma aba"
